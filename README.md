# Smart Expense & Subscription Tracker

A modern finance management platform built with FastAPI, PostgreSQL, Redis, and React.

## Features

- Expense Tracking
- Subscription Management
- Budget Planning
- Analytics Dashboard
- Notifications & Reminders
- Shared Wallets
- Receipt Uploads
- AI Spending Insights

---

## Tech Stack

### Backend

- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Redis
- JWT Authentication

### Frontend

- React
- Redux Toolkit
- Tailwind CSS

### Deployment

- Docker
- Render / Railway
- Vercel
